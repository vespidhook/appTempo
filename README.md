# Gerador de Senha

Aplicativo desenvolvido em React Native(expo) da previsão do tempo

## Instalação

Clone no repositório

```bash
git clone https://github.com/vespidhook/appTempo.git
```
Instale as dependências

```bash
yarn ou npm install
```
Inicie o projeto

```bash
yarn start
```
